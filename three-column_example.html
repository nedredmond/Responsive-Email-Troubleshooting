Some quick tips!
  • Please make copies of the generic templates to edit directly. If you need more/fewer rows, please use the appropriate template.
  • Edit images and text directly; do not copy and paste from another e-mail.
  • Each text area below (Category, Title, Time, Text, and Price) is a different table cell. Please edit these sections individually. If necessary, delete the contents of the cell.
  • The columns below look funky in Display mode. They are formatted to display correctly without the thick table borders that Silverpop adds to help visualize the table formatting. Click "Preview" to see how they will look in an email.
  • Please follow guidelines within images. Resize to 660 or 480 before uploading.
